This guide walks you through installing the SDK, authenticating, and fetching a real-time quote for `700.HK` (Tencent). Pick your language and follow along — the whole flow takes about five minutes.

<Note>
  The SDK uses lazy WebSocket connections — contexts are created synchronously and connect on first use. You do not need to await or callback at construction time.
</Note>

<Steps>
  <Step title="Install the SDK">
    Install the package for your language using your standard package manager.

    <CodeGroup>

    ```bash pip
    pip install longbridge
    ```

    ```bash npm
    npm install longbridge
    ```

    ```toml Cargo.toml
    [dependencies]
    longbridge = "4.0.0"
    ```

    ```xml pom.xml
    <dependencies>
        <dependency>
            <groupId>io.github.longbridge</groupId>
            <artifactId>openapi-sdk</artifactId>
            <version>LATEST</version>
        </dependency>
    </dependencies>
    ```

    ```text C / C++
    Download the prebuilt binary for your platform from the GitHub Releases page:
    https://github.com/longbridge/openapi/releases
    ```

    </CodeGroup>
  </Step>

  <Step title="Set your credentials">
    The SDK reads three environment variables. Set them before running any code.

    ```bash macOS / Linux
    export LONGBRIDGE_APP_KEY="your-app-key"
    export LONGBRIDGE_APP_SECRET="your-app-secret"
    export LONGBRIDGE_ACCESS_TOKEN="your-access-token"
    ```

    ```powershell Windows
    setx LONGBRIDGE_APP_KEY "your-app-key"
    setx LONGBRIDGE_APP_SECRET "your-app-secret"
    setx LONGBRIDGE_ACCESS_TOKEN "your-access-token"
    ```

    You can find these values in your Longbridge user center. For full details on both the legacy API key method and the recommended OAuth 2.0 flow, see the [Authentication](/authentication) page.
  </Step>

  <Step title="Create a config and context">
    Load configuration from your environment variables and create a `QuoteContext`. The context does not connect to the server until you make your first call.

    <Tabs>
      <Tab title="Python">
        ```python
        from longbridge.openapi import Config, QuoteContext

        config = Config.from_apikey_env()
        ctx = QuoteContext(config)
        ```
      </Tab>
      <Tab title="Node.js">
        ```javascript
        const { Config, QuoteContext } = require('longbridge');

        const config = Config.fromApikeyEnv();
        const ctx = QuoteContext.new(config);
        ```
      </Tab>
      <Tab title="Rust">
        ```rust
        use std::sync::Arc;
        use longbridge::{Config, QuoteContext};

        let config = Arc::new(Config::from_apikey_env()?);
        let (ctx, _) = QuoteContext::new(config);
        ```
      </Tab>
      <Tab title="Java">
        ```java
        import com.longbridge.*;
        import com.longbridge.quote.*;

        Config config = Config.fromApikeyEnv();
        QuoteContext ctx = QuoteContext.create(config);
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Fetch a quote">
    Call `quote` (or `getQuote` in Java) with one or more symbols. The SDK connects to the WebSocket endpoint and returns the current price data.

    <Tabs>
      <Tab title="Python">
        ```python
        from longbridge.openapi import Config, QuoteContext

        config = Config.from_apikey_env()
        ctx = QuoteContext(config)
        resp = ctx.quote(["700.HK", "AAPL.US", "TSLA.US", "NFLX.US"])
        print(resp)
        ```
      </Tab>
      <Tab title="Node.js">
        ```javascript
        const { Config, QuoteContext } = require('longbridge');

        async function main() {
          const config = Config.fromApikeyEnv();
          const ctx = QuoteContext.new(config);
          const resp = await ctx.quote(["700.HK", "AAPL.US", "TSLA.US", "NFLX.US"]);
          for (let obj of resp) {
            console.log(obj.toString());
          }
        }

        main();
        ```
      </Tab>
      <Tab title="Rust">
        ```rust
        use std::sync::Arc;
        use longbridge::{Config, QuoteContext};

        #[tokio::main]
        async fn main() -> Result<(), Box<dyn std::error::Error>> {
            let config = Arc::new(Config::from_apikey_env()?);
            let (ctx, _) = QuoteContext::new(config);
            let resp = ctx
                .quote(["700.HK", "AAPL.US", "TSLA.US", "NFLX.US"])
                .await?;
            println!("{:?}", resp);
            Ok(())
        }
        ```
      </Tab>
      <Tab title="Java">
        ```java
        import com.longbridge.*;
        import com.longbridge.quote.*;

        class Main {
            public static void main(String[] args) throws Exception {
                try (Config config = Config.fromApikeyEnv();
                     QuoteContext ctx = QuoteContext.create(config)) {
                    SecurityQuote[] resp = ctx.getQuote(
                        new String[]{"700.HK", "AAPL.US"}).get();
                    for (SecurityQuote obj : resp) {
                        System.out.println(obj);
                    }
                }
            }
        }
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
