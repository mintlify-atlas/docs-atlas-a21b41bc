The Longbridge OpenAPI SDK gives developers programmatic access to real-time market quotes, order management, and portfolio data across Hong Kong, US, and other markets. Whether you're building an automated trading system, a market data pipeline, or an AI-powered investment tool, the SDK provides a consistent, type-safe interface across six languages.

## What you can build

The SDK covers four main capability areas:

- **Trading** — Create, amend, and cancel orders; query today's and historical orders and executions
- **Quotes** — Real-time security quotes, historical candlesticks, option chains, warrants, and market depth
- **Portfolio** — Account balances, stock positions, fund positions, cash flow
- **Real-time push** — WebSocket-based subscriptions for live quote updates and order status changes

## Get started in 4 steps

<Steps>
  <Step title="Install the SDK">
    Pick the package for your language and install it using your package manager.
  </Step>
  <Step title="Authenticate">
    Register an OAuth client or retrieve your API key and secret from the Longbridge user center.
  </Step>
  <Step title="Create a context">
    Instantiate a `QuoteContext` or `TradeContext` with your configuration object.
  </Step>
  <Step title="Make your first call">
    Call `quote(["700.HK", "AAPL.US"])` to fetch live prices, or `submit_order(...)` to place a trade.
  </Step>
</Steps>

<Note>
  The SDK uses a lazy WebSocket connection — contexts are created synchronously and connect to the server on your first API call. No `await` or callback needed at construction time.
</Note>
