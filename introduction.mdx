Longbridge OpenAPI gives developers programmatic access to live market quotes, order management, portfolio data, and real-time push notifications. The SDK is built on a Rust core and exposes idiomatic bindings for Python, Node.js, Java, C, and C++, so you can work in the language that suits your project without sacrificing performance or correctness.

## Supported languages

The SDK is available for six languages, all sharing the same API surface and authentication model:

| Language | Minimum version | Install |
|----------|----------------|---------|
| Rust     | 1.89.0+        | `cargo add longbridge` |
| Python   | 3.8+           | `pip install longbridge` |
| Node.js  | 10+            | `npm install longbridge` |
| Java     | 11+            | Maven Central `io.github.longbridge:openapi-sdk` |
| C        | C99+           | [Download from Releases](https://github.com/longbridge/openapi/releases) |
| C++      | C++17+         | [Download from Releases](https://github.com/longbridge/openapi/releases) |
